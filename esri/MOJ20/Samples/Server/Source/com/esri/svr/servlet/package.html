Provides controller classes for JSP, SimpleEJB, and AdvancedEJB viewers and also Web-based AdvancedEJB admin tool.  This packages all the controller classes for JSP (ServletController for JspServeltViewer), SimpleEJB (EJBControllerI), and AdvancedEJB (EJBControllerII) viewers and also Web-based AdvancedEJB admin tool (CatalogServlet).
