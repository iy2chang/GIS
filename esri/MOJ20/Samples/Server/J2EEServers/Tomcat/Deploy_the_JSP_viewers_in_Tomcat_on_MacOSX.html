Deploy the JSP viewer samples Apache with Tomcat 4 on Mac OS X 

Note:$MOJ refers to the folder in which Map Objects Java Edition Version 2.0 Beta 2 has been installed and $CATALINA_HOME refers to the folder in which the Tomcat has been installed.

  1. Install Map Object Java 2.0.

  2. Install Tomcat 4.1.18 as a Stand-alone Service by following the instructions at http://developer.apple.com/internet/java/tomcat1.html.

  3. Create a folder for MOJ to output the generated images to, and for the JSP viewers to pick up these images from, at '/$CATALINA_HOME/webapps/ROOT/output'.
    Note: To see the /usr/local folder in the Finder, select ‘Go to Folder…’ from the Go menu and enter ‘/usr/local’.

  4. Copy the 'JspViewer' and 'JspServletViewer' directories from the $MOJ/Samples/ServerSamples/JSPViewers/' to the '$CATALINA_HOME/webapps' directory. 

  5. Replace the $MOJ string within the two sample AXL files (usa.axl and world.axl) with the MOJ 2.0 install directory such as ‘/Users/username/MOJ20’

<ARCXML>
...
<WORKSPACES>
<SHAPEWORKSPACE name="ws-0" directory="$MOJ/Samples/Data/USA" />
</WORKSPACES>
...
</ARCXML>
  1. Modify the '$CATALINA_HOME/webapps/JspViewer/WEB-INF/classes/mapservices.xml' and the '$CATALINA_HOME/webapps/JspServletViewer/WEB-INF/classes/mapservices.xml' configuration files so that they look something like this: 

<services>
<name>usa1</name>
...
<output_dir>/usr/local/jakarta-tomcat-4.1.18/webapps/ROOT/output</output_dir>
<output_url></output_url>
<axl_file>$MOJ/Samples/Server/J2EEServers/AXLFiles/usa.axl</axl_file>
...
<name>world2</name>
...
<output_dir>/usr/local/jakarta-tomcat-4.1.18/webapps/ROOT/output</output_dir>
<output_url></output_url> <axl_file>$MOJ/Samples/Server/J2EEServers/AXLFiles/world.axl</axl_file>
...
</services>
  1. Copy all the jar and resources files from the $MOJ/lib folder and $MOJ/Samples/Server/IOStreamProvider/esri_iostreamprovider.jar into the $CATALINA_HOME/shared/lib folder. 

  2. Stop and restart Tomcat.

  3. Open a browser and type one of the following URLs,
    4. http://localhost:8080/JspViewer/viewer.htm or
    http://localhost:8080/JspServletViewer/viewer.htm or
    http://yourlocalmachinename:8080/JspViewer/viewer.htm or
    http://yourlocalmachinename:8080/JspServletViewer/viewer.htm
